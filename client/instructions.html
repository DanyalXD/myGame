Instructions

Controls

  • W A S D - Move Controls
  • R - Ultimate
    • Bruiser ulti - Health Increases for 5 seconds, 20 second cooldown
    • Mage ulti - 360 bullets shot in a circle, 20 second cooldown
  • Right and Left click shoot

Create and Join Game

  • When a player logs in a game is created, code is shown in text box.
  • A player can send the code displayed in the box to another player so they can join their session.
  • A player can paste the code recieved into the box and press join. The players will be in the same session.
  • When a game is over the user will have create game and share the code again.

Game XP

  • A game ends when a player dies
  • The player that kills the other player will gain 20 xp
  • Whoever has the most xp will be 1st on the leaderboard displayed
